# 🧮 Python Calculator Project

سلام! خوش آمدید به پروژه ماشین‌حساب من. این یک پروژه ساده اما کاربردی است که با زبان پایتون توسعه یافته است.

## 📝 توضیحات پروژه (Description)
این پروژه یک ماشین‌حساب است که به کاربر اجازه می‌دهد عملیات ریاضی پایه را انجام دهد. هدف از ساخت این پروژه، تمرین مفاهیم اصلی مثل: Functionها، Loopها، یا Handling Errors] بوده است.

## ✨ ویژگی‌ها (Features)
- [✅] انجام عملیات جمع (`+`)
- [✅] انجام عملیات تفریق (`-`)
- [✅] انجام عملیات ضرب (`*`)
- [✅] انجام عملیات تقسیم (`/`)
- [⭐]تغییر تم ماشین حساب)

## 🛠 تکنولوژی‌های استفاده شده (Technologies Used)
- **Language:** Python 3.x

## 🚀 نحوه اجرا (How to Run)

برای اجرای این پروژه، مراحل زیر را دنبال کنید:

1. ابتدا مطمئن شوید که پایتون روی سیستم شما نصب است.
2. فایل پروژه را دانلود یا Clone کنید
